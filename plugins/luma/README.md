# luma-browser Codex plugin

Install from the repository root:

```bash
# In Codex, add the marketplace path to plugins/luma/.codex-plugin
# or symlink the repo into your Codex plugins directory.
```

This plugin reuses the same `skills/`, `agents/`, `commands/`, and `rules/` as the Cursor and Claude Code packs.

See the root [README.md](../../README.md) for install and agent integration.
